# وَميض (Wameed)

أداة (Framework) + برنامج لتدريب نماذج لغوية **خفيفة** على **الأجهزة الضعيفة**
(معالج فقط، رامة محدودة)، مبنية على فكرة نماذج فضاء الحالة (SSM) مع ثلاث
إضافات خاصة:

1. **وسم الأهمية (Importance Tagging):** لكل حالة درجة أهمية بين 0 و1 — المهم يبقى والتافه يُنسى.
2. **الذاكرة الهرمية (Hierarchical Memory):** أربع طبقات (قصيرة/عاملة/دلالية/مجرّدة) لتذكّر سياق النص الطويل.
3. **العمق الديناميكي (Dynamic Depth):** المدخل السهل يمر بطبقات قليلة، والصعب أعمق.

> الهدف ليس التفوّق المطلق على الترانسفورمر/مامبا، بل **أداة تُنتج نماذج أخفّ
> وأكفأ على الأجهزة الضعيفة تحديداً** — مع توثيق علمي أمين للنتائج (سلبية كانت أم إيجابية).

## الهيكل

```
wameed/        # كود المكتبة (المحرّك): ssm.py + memory.py + utils
scripts/       # سكربتات التدريب والتحقّق والتحليل لكل مرحلة
colab/         # خلايا Colab للتجارب الثقيلة (كارت T4)
results/       # نتائج التجارب (CSV) + ملخّصات JSON
التوثيق/        # توثيق أمين لكل مرحلة بالأرقام الحقيقية
الخطة/          # خطة المشروع التفصيلية مرحلةً مرحلة
STATUS.md      # حالة المشروع الحالية
```

## التشغيل السريع

```bash
python3 -m venv wameed-env && source wameed-env/bin/activate
pip install torch datasets pandas psutil scipy

# المرحلة 1 — وسم الأهمية (اختبار سريع)
python scripts/train_importance.py --mode smoke

# المرحلة 2 — الذاكرة الهرمية: تحقّق + تدريب
python scripts/verify_memory.py
python scripts/train_memory.py --mode smoke
```
