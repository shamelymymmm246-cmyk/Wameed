## 🧠 خلفية مبسّطة + التعريف الدقيق

### ضغط المعاني (Memory Compression)

**المشكلة:** متجهات الذاكرة (M1, M2, M3, M4) تأخذ حجماً من الرامة.
كلما زاد state_dim أو عدد الطبقات، زادت الرامة.

**الحل — خفض البُعد بإسقاط خطي (Projection):**
```
M_compressed = tanh(W_compress · M_original)
```
حيث W_compress مصفوفة أوزان `(D_small, D_original)`.

- **مثال:** D_original = 128 → D_small = 32 (ضغط 4:1 = 75% توفير رامة).
- **التكلفة:** قد تفقد بعض المعلومات → ترتفع PPL قليلاً.
- **الهدف:** حصل على ≥30% توفير رامة مقابل ≤5% ارتفاع PPL.

**جدول الضغط المخطَّط له:**

| نسبة الضغط | D_original | D_compressed | توفير رامة متوقّع |
|-----------|-----------|--------------|-----------------|
| بلا ضغط (baseline) | 128 | 128 | 0% |
| 2:1 | 128 | 64 | ~50% في الذاكرة |
| 4:1 | 128 | 32 | ~75% في الذاكرة |
| 8:1 | 128 | 16 | ~87% في الذاكرة |

**ملاحظة:** التوفير الفعلي أقل من 75% لأن باقي النموذج لا يتغيّر.
لكن التوفير في الذاكرة الهرمية تحديداً ملموس جداً.

---

### العمق الديناميكي (Dynamic Depth / Early-Exit)

**المشكلة:** كل توكن يمرّ بنفس عدد الطبقات — حتى لو كان توقّعه سهلاً.

**الحل:** بعد كل طبقة، يقرّر النموذج: «هل أنا واثق كفاية؟»
```
ثقة_t = sigmoid(W_exit · h_t + b_exit)
إذا ثقة_t > عتبة_الخروج: اخرج الآن (skip الطبقات الباقية)
```

**مثال تفسيري:**
- جملة: "Paris is the capital of ___"
  - التوكنات السهلة ("is", "the") → ثقة عالية بعد طبقة 1 → خروج مبكر.
  - التوكن الصعب ("France") → ثقة منخفضة → يكمل لطبقات أعمق.

**كيف يتعلّم رأس الثقة؟**
```python
# إشارة التدريب: هل التنبؤ كان صحيحاً في هذه الطبقة؟
pred_at_layer = output_proj(h_at_layer)    # تنبؤ من الطبقة الحالية
was_correct = (pred_at_layer.argmax(-1) == target).float()
exit_target = was_correct  # 1 = كنت صح، يجب أن أخرج. 0 = خطأ، أكمل.

# خسارة رأس الخروج
exit_loss = BCE(predicted_confidence, exit_target)
total_loss = main_loss + 0.1 * exit_loss  # معامل صغير
```

**الخطوات التفصيلية:**

**أ — أضف خيار الضغط لـ `HierarchicalMemory`:**

```python
class HierarchicalMemory(nn.Module):
    def __init__(self, state_dim, compress_ratio=1, ...):
        ...
        # حجم المخزّن بعد الضغط
        self.comp_dim = state_dim // compress_ratio
        
        if compress_ratio > 1:
            # طبقات الضغط والفكّ
            self.compress = nn.Linear(state_dim, self.comp_dim)
            self.decompress = nn.Linear(self.comp_dim, state_dim)
        
        # أحجام الذاكرة بعد الضغط
        self.M3 = torch.zeros(batch_size, self.comp_dim)
        self.M4 = torch.zeros(batch_size, self.comp_dim // 2)
    
    def _compress(self, x):
        """يضغط متجه الذاكرة"""
        if self.compress_ratio > 1:
            return torch.tanh(self.compress(x))
        return x
    
    def _decompress(self, x):
        """يفكّ الضغط للقراءة"""
        if self.compress_ratio > 1:
            return torch.tanh(self.decompress(x))
        return x
```

### المهمة 3.2 — مصنّف صعوبة بسيط للعمق الديناميكي

**الشرط المسبق:** يجب أن يكون النموذج بـ ≥ 3 طبقات.
على جهازه الحالي (نموذج بطبقتين) يمكن إضافة طبقة ثالثة على Colab.

**الخطوات:**

```python
class DynamicDepthSSM(nn.Module):
    def __init__(self, ..., num_layers=4, exit_threshold=0.8):
        super().__init__()
        self.num_layers = num_layers
        self.exit_threshold = exit_threshold
        
        # رأس الثقة لكل طبقة
        self.exit_heads = nn.ModuleList([
            nn.Linear(state_dim, 1) for _ in range(num_layers - 1)
            # الطبقة الأخيرة لا تحتاج رأس خروج
        ])
        
        # إسقاط التنبؤ المبكر (لكل طبقة وسيطة)
        self.early_exit_projs = nn.ModuleList([
            nn.Linear(state_dim, vocab_size) for _ in range(num_layers - 1)
        ])
    
    def forward(self, x, targets=None):
        h = self.embedding(x)
        
        exit_losses = []
        layers_used = []   # لتتبّع الطبقات المستخدمة
        
        for layer_idx, layer in enumerate(self.layers):
            h = layer(h)   # طبقة SSM
            
            if layer_idx < self.num_layers - 1:  # لكل طبقة عدا الأخيرة
                # احسب الثقة
                confidence = torch.sigmoid(self.exit_heads[layer_idx](h))
                
                # هل نخرج مبكراً؟ (أثناء التقييم فقط)
                if not self.training and confidence.mean() > self.exit_threshold:
                    layers_used.append(layer_idx + 1)
                    logits = self.early_exit_projs[layer_idx](h)
                    return logits, layers_used
                
                # أثناء التدريب: احسب خسارة رأس الخروج
                if targets is not None:
                    early_logits = self.early_exit_projs[layer_idx](h)
                    correct = (early_logits.argmax(-1) == targets).float()
                    exit_loss = nn.functional.binary_cross_entropy(
                        confidence.squeeze(-1), correct
                    )
                    exit_losses.append(exit_loss)
        
        layers_used.append(self.num_layers)
        logits = self.output_proj(h)
        
        # الخسارة الكلية
        main_loss = self.criterion(logits, targets)
        exit_loss_total = sum(exit_losses) * 0.1 if exit_losses else 0
        
        return logits, main_loss + exit_loss_total, layers_used
```

**معيار القبول (بالأرقام):**

- على مدخلات سهلة (توكنات شائعة متوقّعة)، متوسط الطبقات المستخدمة
  **أقل** منه على الصعبة.
- الفرق يجب أن يكون واضحاً: مثلاً 1.5 vs 3.2 طبقة.

### المهمة 3.3 — قياس الحساب الموفّر على المدخلات السهلة

```python
def benchmark_dynamic_depth(model, val_loader):
    """يحسب التوفير في الحساب من الخروج المبكر"""
    total_tokens = 0
    total_layers = 0
    times_with_dd = []
    times_without_dd = []
    
    for batch in val_loader:
        # قياس الزمن بالعمق الديناميكي
        t0 = time.time()
        _, _, layers_used = model(batch, use_dynamic_depth=True)
        t1 = time.time()
        times_with_dd.append(t1 - t0)
        
        # قياس الزمن بدونه (يمرّ بكل الطبقات)
        t0 = time.time()
        _ = model(batch, use_dynamic_depth=False)
        t1 = time.time()
        times_without_dd.append(t1 - t0)
        
        total_layers += sum(layers_used)
        total_tokens += len(layers_used)
    
    avg_layers = total_layers / total_tokens
    max_layers = model.num_layers
    skip_pct = (1 - avg_layers / max_layers) * 100
    speedup = np.mean(times_without_dd) / np.mean(times_with_dd)
    
    print(f"متوسط الطبقات المستخدمة: {avg_layers:.2f} / {max_layers}")
    print(f"نسبة الحساب الموفّرة: {skip_pct:.1f}%")
    print(f"تسريع الزمن: {speedup:.2f}x")
    return skip_pct, speedup
```

## 🔧 الخطط البديلة (مشاكل المرحلة 3)

### البديل أ — الضغط يضرّ الجودة بشدّة

**الأعراض:** PPL ترتفع > 10% عند أي نسبة ضغط.

**الحل التدريجي:**
1. قلّل نسبة الضغط: 8:1 → 4:1 → 2:1 حتى يصبح الأثر مقبولاً.
2. أضف طبقة تطبيع (LayerNorm) قبل وبعد الضغط.
3. استخدم تنشيط أكثر قدرة: `GELU` بدل `tanh`.
4. إذا 2:1 ما زال يضرّ: سجّل النتيجة بأمانة، الضغط البسيط لا يكفي لهذا النموذج.

---

### البديل ب — الخروج المبكر يخرج دائماً (بعد طبقة 1)

**الأعراض:** avg_layers ≈ 1.0 لكل التوكنات.

**الحل:** ارفع العتبة: `exit_threshold = 0.8 → 0.9 → 0.95`

---

### البديل ج — الخروج المبكر لا يخرج أبداً (يكمل كل الطبقات)

**الأعراض:** avg_layers = num_layers لكل التوكنات.

**الحل:**
1. خفّض العتبة: `exit_threshold = 0.8 → 0.7 → 0.5`
2. تحقّق من أن رأس الثقة تدرّب: طباعة الثقة لتوكنات متنوّعة.
3. قد يحتاج رأس الثقة دورات تدريب أكثر.

---

### البديل د — النماذج الصغيرة (طبقتان) لا تستفيد من Dynamic Depth

**هذا متوقّع** على الجهاز المحدود.

**الحل:**
- وثّق المبدأ نظرياً: «على نموذج بـ 4+ طبقات، الخروج المبكر يوفّر X%.»
- أثبته على Colab بنموذج 4 طبقات.
- لا تبالغ في الادّعاء على النموذج الصغير.
